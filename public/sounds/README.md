# 사운드 파일 안내

이 폴더에 `click.mp3` 파일을 추가해주세요.

## 파일 요구사항
- 파일명: `click.mp3`
- 형식: MP3 오디오 파일
- 권장 길이: 0.1~0.3초 (짧은 클릭/팝 사운드)
- 권장 볼륨: 적당한 볼륨 (앱에서 0.3으로 조절됨)

## 무료 사운드 리소스
- [Freesound.org](https://freesound.org) - "click" 또는 "pop" 검색
- [Zapsplat](https://www.zapsplat.com) - 무료 사운드 효과
- [Mixkit](https://mixkit.co/free-sound-effects/click/) - 무료 클릭 사운드
